# table for veeery big data.

## Task

The table accepts> 10,000 thousand items as input. It is necessary to optimize data output and scroll inside the table

## Development stack

The following tools were used for this project.

- React (hooks)
- TypeScript

## Run project

Install packages

```sh
$ yarn
```

Run dev server

```sh
$ yarn start
```
